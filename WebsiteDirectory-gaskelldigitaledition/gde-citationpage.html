Using The Elizabeth Gaskell Journal: Digital Edition

Thank you for visiting The Elizabeth Gaskell Journal: Digital Edition website! This edition was prepared as a digital dissertation project at Michigan State University in 2020, and continues to be updated. The code is available via a GitHub Repository.

Please note:
The text of Elizabeth Gaskell's journal is still under copyright in England.
